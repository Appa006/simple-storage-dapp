# 💾 Simple Storage DApp (Лабораторная №14)

Децентрализованное приложение для записи и чтения текстовой информации в блокчейне Ethereum (тестовая сеть Sepolia). Проект демонстрирует базовые принципы CRUD-взаимодействия со смарт-контрактом.

![Интерфейс приложения](images/screenshot.png)

## 📝 О проекте
Это минималистичный DApp, который позволяет пользователю:
1. **Подключить MetaMask** для идентификации в сети.
2. **Считать данные** из переменной `storedData` смарт-контракта.
3. **Сохранить новую строку** в блокчейн через вызов функции `set`.

## 🛠 Технологии
* **Web3 Library:** [Ethers.js v6.10](https://docs.ethers.org/v6/)
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Network:** Ethereum Sepolia Testnet
* **Smart Contract:** `0x1A94AcF67882c7599B93A9a6E282cdFa59aB36c3`

## ⚙️ Требования для работы
Чтобы успешно взаимодействовать с блокчейном через этот интерфейс, вам понадобятся:
* Браузер с установленным расширением **MetaMask**.
* Тестовые токены **Sepolia ETH** на балансе (их можно получить бесплатно в "Sepolia Faucet").
* Локальный веб-сервер для запуска (например, расширение **Live Server** в VS Code).
